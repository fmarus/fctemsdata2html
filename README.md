# fctemsdata2html
fctemsepdata2html.py - FortiClient EMS tool to import Endpoint Data and extract it into HTML format.

## Installation
* Clone the repo, or download fctemsdata2html.py and requirements.txt
* ```pip3 install -r requirements.txt```
